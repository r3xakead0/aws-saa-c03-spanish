Simulación AWS SAA-C03

1. Una empresa tiene una aplicación que se ejecuta en un gran tipo de instancia de Amazon EC2 de propósito general que forma parte de un grupo EC2 Auto Scaling. La compañía quiere reducir costos futuros asociados a esta aplicación. Después de que la compañía revisa las métricas y los registros en Amazon CloudWatch, la compañía nota que esta aplicación se ejecuta aleatoriamente un par de veces al día para recuperar y administrar datos. Según CloudWatch, el tiempo de ejecución máximo para cada solicitud es de 10 minutos, el uso de memoria es de 4 GB y las instancias siempre están en estado de ejecución. ¿Qué solución reducirá los costos MÁS?

2. Una empresa de eventos en línea ejecuta una aplicación de venta de boletos electrónicos para conciertos de música en vivo. La demanda es impredecible y puede ser particularmente alta para los artistas populares. ¿Cuál sería la forma más óptima de alojar el contenido estático para la aplicación web frond-end? El sitio es rico en archivos de audio y video de actuaciones.

3. Una compañía de medios está diseñando una nueva solución para renderizado gráfico. La aplicación requiere hasta 400 GB de almacenamiento para datos temporales que se descartan después de renderizar las tramas. La aplicación requiere aproximadamente 40,000 IOPS aleatorias para realizar el renderizado. ¿Cuál es la opción de almacenamiento MÁS rentable para esta aplicación de renderizado?

4. Un arquitecto de soluciones está trabajando para optimizar una aplicación de administración de documentos heredada que se ejecuta en varios servidores Microsoft Windows en un centro de datos local. El director de información quiere migrar el almacenamiento local a AWS, pero aún así debe ser capaz de soportar la aplicación heredada. ¿Qué debe hacer el arquitecto de soluciones para cumplir con estos requisitos?

5. Una empresa ha decidido utilizar AWS para lograr una alta disponibilidad. La compañía utiliza métricas y alarmas de Amazon CloudWatch para monitorear la arquitectura. Un arquitecto de soluciones se da cuenta de que la compañía no es capaz de lanzar algunas instancias. El arquitecto de soluciones encuentra el siguiente mensaje: CUOTA EC2 EXCEDIDA. ¿Cómo puede el arquitecto de soluciones garantizar que la compañía pueda lanzar todas las instancias EC2 correctamente?

6. Una empresa espera que la base de usuarios de su aplicación de compras en el hogar aumente a 3 veces el nivel actual en 1 año. La compañía aloja la aplicación en una región de AWS. Un arquitecto de soluciones debe recomendar cambios de diseño para apoyar el crecimiento esperado y reducir la sobrecarga operativa. ¿Qué cambios de diseño cumplen con estos requisitos?
