Simulación AWS SAA-C03

1. Una empresa tiene una aplicación que se ejecuta en un gran tipo de instancia de Amazon EC2 de propósito general que forma parte de un grupo EC2 Auto Scaling. La compañía quiere reducir costos futuros asociados a esta aplicación. Después de que la compañía revisa las métricas y los registros en Amazon CloudWatch, la compañía nota que esta aplicación se ejecuta aleatoriamente un par de veces al día para recuperar y administrar datos. Según CloudWatch, el tiempo de ejecución máximo para cada solicitud es de 10 minutos, el uso de memoria es de 4 GB y las instancias siempre están en estado de ejecución. ¿Qué solución reducirá los costos MÁS?

2. Una empresa de eventos en línea ejecuta una aplicación de venta de boletos electrónicos para conciertos de música en vivo. La demanda es impredecible y puede ser particularmente alta para los artistas populares. ¿Cuál sería la forma más óptima de alojar el contenido estático para la aplicación web frond-end? El sitio es rico en archivos de audio y video de actuaciones.

3. Una compañía de medios está diseñando una nueva solución para renderizado gráfico. La aplicación requiere hasta 400 GB de almacenamiento para datos temporales que se descartan después de renderizar las tramas. La aplicación requiere aproximadamente 40,000 IOPS aleatorias para realizar el renderizado. ¿Cuál es la opción de almacenamiento MÁS rentable para esta aplicación de renderizado?

4. Un arquitecto de soluciones está trabajando para optimizar una aplicación de administración de documentos heredada que se ejecuta en varios servidores Microsoft Windows en un centro de datos local. La aplicación accede a una gran cantidad de archivos en un recurso compartido de archivos de red que se está quedando sin espacio. El director de información quiere migrar el almacenamiento local a AWS, pero aún así debe ser capaz de soportar la aplicación heredada. ¿Qué debe hacer el arquitecto de soluciones para cumplir con estos requisitos?

5. Una empresa ha decidido utilizar AWS para lograr una alta disponibilidad. La arquitectura de la compañía consiste en un balanceador de carga de aplicaciones frente a un grupo de Auto Scaling que consta de instancias de Amazon EC2. La compañía utiliza métricas y alarmas de Amazon CloudWatch para monitorear la arquitectura. Un arquitecto de soluciones se da cuenta de que la compañía no es capaz de lanzar algunas instancias. El arquitecto de soluciones encuentra el siguiente mensaje: CUOTA EC2 EXCEDIDA. ¿Cómo puede el arquitecto de soluciones garantizar que la compañía pueda lanzar todas las instancias EC2 correctamente?

6. Una empresa espera que la base de usuarios de su aplicación de compras en el hogar aumente a 3 veces el nivel actual en 1 año. La compañía aloja la aplicación en una región de AWS. La compañía utiliza una instancia de base de datos de Amazon RDS para MySQL, un balanceador de carga de aplicaciones y Amazon Elastic Container Service (Amazon ECS) para alojar el sitio web, el contenido estático y los microservicios. Un arquitecto de soluciones debe recomendar cambios de diseño para apoyar el crecimiento esperado y reducir la sobrecarga operativa. ¿Qué cambios de diseño cumplen con estos requisitos? Están explorando formas en las que sus analistas de negocios pueden ejecutar consultas en millones de reclamos y mostrar gráficos en paneles de inteligencia empresarial (BI). ¿Qué servicios recomendarías para lograrlo con el menor esfuerzo administrativo?

8. Una aplicación que se ejecuta en su instancia de Amazon EC2 en una VPC requiere acceso de solo lectura a los archivos almacenados en un bucket de Amazon S3. ¿Cuáles son las dos acciones necesarias para implementar esto de manera segura? SELECCIONAR 2

9. Una tienda de mascotas en línea tiene un sistema crítico de vacunación para mascotas que actualmente se ejecuta en una sola instancia EC2. Se le ha pedido que ponga la solución altamente disponible con el menor esfuerzo administrativo. ¿Qué pasos recomendarías? SELECCIONAR 3

10. Una empresa edu-tech almacena documentos de examen de estudiantes terminados en un cubo de Amazon S3. La compañía ha adoptado una estrategia basada en la nube y está experimentando un crecimiento significativo. Su producto también ha sido lanzado en múltiples países. Deben apegarse a las regulaciones locales que les exigen almacenar los escaneos durante 6 años, pero se requiere eliminarlos posteriormente ya que los datos se consideran sensibles. ¿Qué característica de Amazon S3 les permitirá lograr esto?

11. Una startup de AI/ML está buscando un servicio de AWS para almacenar metadatos simples y estructurados en imágenes recopiladas utilizando una base de datos especialmente diseñada. Su aplicación utiliza actualmente una base de datos MongoDB autogestionada. El CTO ha ordenado que todas las implementaciones de AWS no tengan servidor, y se ha reservado tiempo de desarrollo para reelaborar cualquier API según sea necesario. ¿Qué servicio de AWS recomendarías?

12. Un desarrollador almacenó un archivo de configuración de aplicación importante en /app, pero el archivo ya no existe después de reiniciar la instancia de Linux de Amazon EC2. ¿Qué declaraciones describen posibles razones de su desaparición? SELECCIONAR 3

13. Una clínica de mascotas quiere utilizar Amazon Rekognition para identificar de inmediato al animal detectado en una foto proporcionada por su cliente, para que la cita se pueda hacer con el veterinario adecuado. ¿Cuál será la solución más sencilla?

14. Trabajas para el CIO de una compañía de seguros de automóviles. Recientemente, sus competidores han informado que fueron sometidos a ataques de inyección SQL por parte de actores maliciosos. Al CIO le gustaría proteger su infraestructura de amenazas similares. Sus servidores web se ejecutan detrás de un balanceador de carga de aplicaciones (ALB). ¿Qué recomendarías?

15. El equipo de ingeniería de aplicaciones ha adoptado una estrategia de solo servidor. Se ha solicitado al equipo de seguridad empresarial que evite el uso de instancias EC2 solo en las cuentas de AWS de los equipos de aplicaciones. Sin embargo, los desarrolladores de aplicaciones deben seguir siendo administradores de sus cuentas de AWS. ¿Qué enfoque recomendarías? SELECCIONAR 2
